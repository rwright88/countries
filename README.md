# countries

Country datasets

## Datasets structure

One row per country per year, with these columns:

- country_code
- country_name
- year
- statistics (any number of columns)

## Completed datasets

Dataset      | Source
-------------|-------
companies    | [Forbes](https://www.forbes.com/global2000/list/)

obesity      | [WHO](https://ourworldindata.org/obesity)
